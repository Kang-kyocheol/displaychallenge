속닥존 디스플레이

화면은 함께 보고, 소리는 각자 또렷하게. 사용자 위치에 맞춰 한국어·영어·개인 안내 음성을 각각 전달하는 다중 음향영역 프라이버시 디스플레이입니다.

대표 영상 핵심 기능 사용 시나리오
속닥존 디스플레이 대표 영상 재생 버튼을 눌러 확인하세요
OVERVIEW

프로젝트 개요

아이디어가 해결하고자 하는 문제와 제품의 핵심 가치를 소개합니다.

병원·공항·은행과 같은 공공 공간에서 안내 음성이 주변 사람에게 그대로 노출되거나 여러 기기의 소리가 섞이는 문제를 해결합니다. 센서가 사용자 위치를 인식하고, 각 위치에 필요한 음성만 집중적으로 전달합니다.
CORE FEATURES

핵심 기능

사용자가 실제로 체감할 수 있는 주요 기능입니다.

01

사용자 위치 추적

카메라 또는 ToF 센서로 화면 앞 사용자들의 위치와 거리를 인식합니다.

02

개인 음향영역 형성

다채널 스피커와 음장 제어로 좌우 사용자에게 서로 다른 음성을 전달합니다.

03

다국어·프라이버시 안내

사용자별 언어를 제공하고 주변으로의 소리 누설을 최소화합니다.

SCENARIO

사용 흐름

실제 환경에서 제품이 작동하는 과정을 단계별로 보여줍니다.

1

사용자 접근

센서가 사용자 A와 B의 위치를 인식합니다.

2

언어·업무 선택

각 사용자가 원하는 언어와 안내 항목을 선택합니다.

3

개별 음성 전달

A에게는 한국어, B에게는 영어 안내를 전달합니다.

4

주변 누설 억제

사용자 사이와 주변에서는 안내음이 작게 들리도록 제어합니다.

IMPACT

기대효과

사용자 가치와 산업적·사회적 확장 가능성입니다.

공공·의료

병원 접수, 민원 창구, 공항 안내에 적용할 수 있습니다.

차량·이동공간

차량, 셔틀, 열차 좌석용 개인 오디오로 확장할 수 있습니다.

문화·교육

박물관, 전시장, 스마트 교실에서 다국어 안내를 제공합니다.
