ExitFlow

화재·연기·혼잡 상황을 반영하여 현재 가장 안전한 대피 경로를 바닥과 벽 하단에 실시간으로 표시하는 스마트 비상구 디스플레이입니다.

대표 영상 핵심 기능 사용 시나리오
ExitFlow 대표 영상 재생 버튼을 눌러 확인하세요
OVERVIEW

프로젝트 개요

아이디어가 해결하고자 하는 문제와 제품의 핵심 가치를 소개합니다.

기존 비상구 표지는 항상 같은 방향만 알려주지만 재난 상황에서는 기존 경로가 위험해질 수 있습니다. ExitFlow는 센서 데이터를 분석해 위험구역을 차단하고 지금 이동해야 할 가장 안전한 경로를 안내합니다.
CORE FEATURES

핵심 기능

사용자가 실제로 체감할 수 있는 주요 기능입니다.

01

위험구역 실시간 판단

연기·온도·유해가스·인원 밀집도 데이터를 종합해 위험 구간을 탐지합니다.

02

대피 방향 자동 변경

안전 출구와 우회 경로를 계산하고 바닥 화살표를 즉시 전환합니다.

03

교통약자 맞춤 경로

계단을 제외하고 경사로와 피난공간 중심의 별도 경로를 제공합니다.

SCENARIO

사용 흐름

실제 환경에서 제품이 작동하는 과정을 단계별로 보여줍니다.

1

재난 감지

화재·연기·혼잡 상황을 센서가 감지합니다.

2

위험구역 차단

분석 시스템이 위험 구간을 진입 금지 구역으로 설정합니다.

3

안전 경로 계산

대피자 위치와 출구 상태를 반영해 경로를 재계산합니다.

4

디스플레이 전환

초록 화살표, 빨간 X, 노란 경고와 접근성 경로를 표시합니다.

IMPACT

기대효과

사용자 가치와 산업적·사회적 확장 가능성입니다.

사용자 안전

낮은 시야에서도 대피 방향을 직관적으로 인식할 수 있습니다.

스마트빌딩

쇼핑몰·병원·학교·공항의 안전 인프라로 확장할 수 있습니다.

포용적 대피

교통약자의 이동 조건을 고려한 별도 경로를 제공합니다.
